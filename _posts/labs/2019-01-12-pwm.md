## for loop!

### for loops with range()

```python
for i in range(5):
  print(i)
```

```python
for i in range(5, 10):
  print(i)
```

```python
for i in range(0,15,3):
   print(i)
```

```python
for i in range(1024,-1,-1):
   print(i)
```
